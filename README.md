# 記帳系統 (Accounting System) 📊

這是一個使用 Python 開發的記帳系統，能幫助你管理支出與收入，並生成月度收支報告！💰

---

## 功能特色 ✨

- **新增支出**  
  輸入日期、類別、金額、描述，並自動儲存到 Excel 檔案。

- **新增存款**  
  更新總收入數據，並儲存在同一個 Excel 檔案中。

- **生成月度圖表**  
  根據指定月份生成圓餅圖，直觀展示收入、支出與盈餘。

- **中文支援**  
  使用 Heiti TC 字體，確保圖表中文字正常顯示。

---
